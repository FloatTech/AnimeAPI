# AnimeAPI
二次元相关API，详细说明可前往各文件夹查看。
## aireply
调用 AI 回复对话
## ascii2d
ascii2d 搜图
## baidutts
百度文字转语音
## classify
AI图片打分
## danbooru
DeepDanbooru 二次元图片标签识别
## mockingbird
拟声 TTS
## nsfw
图片合规性审查
## pixiv
P站解析与图片下载
## saucenao
SauceNao 搜图
## scale
采用[叔叔的模型](https://github.com/bilibili/ailab)放大二次元图片
## shindanmaker
基于 https://shindanmaker.com 的 API
## yandex
Yandex 搜图
